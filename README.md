# Apartment Storage Management System (ASMS)

A modern **Apartment Storage Rental and Management Platform** that enables residents to explore, rent, and manage storage units within apartment complexes through **3D visualization, real-time availability tracking, and online booking**.

# Project Overview

ASMS (3D-Simulated Apartment Storage Management System) is a fullstack platform designed to **digitize storage rental services inside apartment buildings**.

Traditional storage rental in apartment complexes often lacks transparency and digital accessibility. This project addresses that issue by providing a modern platform where users can:

- visually explore storage units
- check real-time availability
- book storage spaces online
- manage reservations and payments digitally

The system focuses on **usability, transparency, and modern digital infrastructure for property services**.

# Key Features

## 3D Storage Visualization

Users can explore available storage spaces using **3D visualization**, allowing them to understand the size, layout, and location of each unit before renting.

## Real-time Storage Availability

The system provides **live availability updates**, ensuring users always see accurate information about storage unit status.

## Online Booking System

Users can reserve storage units directly through the platform with a streamlined booking process.

## Online Payment Integration

The platform integrates **PayOS** for secure digital payments, allowing users to complete bookings instantly.

## Google Maps Integration

Users can easily locate storage areas and apartment buildings through **Google Maps integration**.

## Admin Management Dashboard

Administrators can manage:

- storage units
- bookings
- payment records
- user accounts
- availability updates

through a centralized management dashboard.

# System Architecture

The system follows a modern **web-based layered architecture**.
Frontend (React / React Native)
↓
REST API
↓
Backend (.NET / C#)
↓
SQL Server Database

Additional services:

- Cloudinary → media storage
- PayOS → payment processing
- Google Maps API → location services
- MailKit → email notification service
